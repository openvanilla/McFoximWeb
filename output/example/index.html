使用說明

  • 建議使用電腦打開這個網頁，或是將您的手機或平板接上實體鍵盤，手機輸入則請使用英語鍵盤
  • 在打字的時候，會主動查詢可以自動補完的候選字
  • 使用英文字母—以及標點輸入文字時，會進入輸入區
  • 用 Tab 或是 1-9 鍵選擇要補完候選字
  • 出現可以自動補完候選字時，可以用上下鍵移動游標，用上下頁按鍵（Page Up/Page Down）換頁
額外說明

McFoxIM 的名稱來自於 ISO 639-5 語言代碼，fox 就是台灣南島民族語言的代號，IM 則是 Input Method 的縮寫，合起來就是 McFoxIM。

Windows、macOS、iOS、Android 等平台，可以參考原語會的官方族語輸入法。
